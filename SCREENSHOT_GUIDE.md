## üé® Screenshot Best Practices

### Before Taking Screenshots

1. **Clean Browser**
   - Clear cache
   - Close unnecessary tabs
   - Use incognito/private mode
   - Full screen mode (F11)

2. **Test Data**
   - Use realistic names
   - Use proper values
   - Show varied scenarios
   - Avoid "test" or "demo" in visible text

3. **Timing**
   - Capture during demo mode
   - Show active states
   - Capture animations mid-way
   - Show loading states

### During Screenshot

1. **Resolution**
   - Use 1920x1080 or higher
   - Capture full window
   - Include browser chrome (optional)
   - Consistent sizing

2. **Content**
   - Center important elements
   - Show complete features
   - Avoid cut-off text
   - Include context

3. **Quality**
   - PNG format (lossless)
   - High quality
   - No compression artifacts
   - Clear text

### Tools

**Windows**:
- `Win + Shift + S` - Snipping Tool
- `Win + PrtScn` - Full screen
- Snagit (professional)
- ShareX (free, powerful)

**Browser Extensions**:
- Awesome Screenshot
- Nimbus Screenshot
- Full Page Screen Capture

- Loom (easy, cloud-based)

   - Show vitals updating

   - Show emergency system

- 30 FPS minimum
- MP4 format
